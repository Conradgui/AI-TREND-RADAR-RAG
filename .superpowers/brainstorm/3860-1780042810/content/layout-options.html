右侧滑出面板

点击 header 的 "AI Agent" 按钮，右侧面板从右侧滑出覆盖在内容区上。不占用原有布局空间，平时隐藏不干扰看报告。

类似 VS Code 终端，对话面板从底部弹出，可调整高度。输入框在底部，历史消息向上滚动。
